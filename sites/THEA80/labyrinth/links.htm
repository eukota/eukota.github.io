Labyrinth sites that might be of interest:

Labyrinth Ring Home Page - This is the master site for all of the pages that are registered in the Labyrinth webring. The best page to start at when looking for a lot of different Labyrinth sites.

The Labyrinth Complete Script - This is a very simple text - based site which contains the entire script for the movie.

Labyrinth - The Movie - A well done site with a little bit of everything on it, including a Sir Didymus page.

Save The Labyrinth - A web campaign site to get Labyrinth rereleased on the big screen. Interesting points and idea, but the page has not been updated in a while.

The Gallery of Brian Froud - Images as well as other links to the Dark Crystal and other work that Brian Froud, chief Costume Designer and conceptual designer for The Dark Crystal has done.

The Dark Crystal - A link to this site's sister site on the Dark Crystal.
